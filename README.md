# 生成项目
- npx create-react-app react_demo
- cd my-app
- npm start

# 项目目录
- public 网页入口文件index.html存放位置
- node_modules 各个插件存放位置
- src
  - actions 存放react action路径
  - api 存放接口文件
  - common 存放公共文件
  - components
  - images 存放图片
  - router
    - config.js 配置路由
    - index.js  将路由与组件对应，这样在跳转到某路径时，会渲染对应的组件
  - stores 存放 react action具体执行方法
  - styles 存放样式文件
  - views
  - index.js react入口文件

# 安装插件
- yarn add react-router-dom
